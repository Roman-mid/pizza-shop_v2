## React Pizza Shop


### Key Features
- Add and remove items from the cart </br>
- Adjust item quantities in the cart </br>
- Search items by name </br>
- Filter and display products by category </br>


### Tech Stack
- React </br>
- Redux Toolkit </br>
- TypeScript </br>
- React Router (BrowserRouter) </br>
- ContentLoader </br>
- CSS / SCSS </br>
